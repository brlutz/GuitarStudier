How to drill sight reading:

Guitar sight reading is an important skill to practice. It has two important purposes, first one is to learn the fretboard. Learning the fretboard allows you to find chord tones when soloing. The second is to be able to read sheet music. Being able to read sheet music will help you learn how to play lead music that may not be written specifically for guitar. You can learn vocal, saxaphone, or any other melodic music!

Practice Tips:

At first, just practice finding each note. As you become more comfortable, take note of the notes you miss and take the time to practice them specifically. For an added challange, try to set a metronome and play each note on the beat. Gradually speed this up until you're a sight reading pro!

Set yourself a goal in exercises or time. Something like 50 four note sections or 30 minutes a day. Set yourself a schedule, get some free time and practice without distractions. You'll start slow at first, but slowly, with practice, will increase the speed you're able to play these. Getting fast will take practice, but that's all it takes! Practice consistently and you'll see consistent results.
